# Challenge: Remove Duplicates

## Instructions

Write a function called `removeDuplicates`. Your function should receive an `array` as an argument and return a new array with any duplicate values removed. The new array should be in the same order as the original array

### How to complete the challenge
- `npm install` to get Jest installed
- Add your solution to `challenge-code.js`
- Run the provided unit test. It should pass. `npm test`
- To run your function in the terminal and check the output, run `node index.js`

### Function Signature

```javascript
const removeDuplicates: any[] = ( array: any[] ) => {}
```

### Example usage

```javascript
removeDuplicates( [ 1, 2, 2, 3, 3, 1, 4, 5 ] );
// [ 1, 2, 3, 4, 5 ]
```
